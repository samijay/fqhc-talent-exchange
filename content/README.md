# Content Research & Drafts

This folder is for research materials that feed into blog articles, page content, and SEO optimization.

## Folder Structure

- **research/salary-surveys/** — Drop salary survey PDFs, reports, and data here
- **research/program-descriptions/** — ECM, CCM, CalAIM program docs, DHCS materials
- **research/competitor-articles/** — Articles from competitors or publications you want to match in quality/style
- **research/resume-templates/** — Resume templates, screening question frameworks, and career guidance docs
- **drafts/** — Working drafts before they go into the Next.js blog

## How to Use

1. Drop reference materials into the appropriate research/ subfolder
2. In Claude.ai, create a Project called "FQHC Content" and upload the same materials
3. Use Claude.ai for drafting and ideation (it has your brand context from the Project)
4. Bring finished drafts to Cowork to wire into the Next.js blog with SEO metadata
